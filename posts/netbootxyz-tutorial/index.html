Booting to network

To boot to the network youâ€™ll need a BIOS and NIC that supports it

  • enable in BIOS
    • enable EFI PXE Boot
    • enable Legacy (BIOS) PXE Boot
  • Figure out boot override or network boot key
  • Power on and boot to network (BIOS of EFI)

See the boot menu, choose OS and go!

Word of caution, there might be some that do not work. This is a moving target. e.g. Ubuntu 23.10 isnâ€™t working for me now, but could soon. Other OS are fine. You may need to try different NICs if you are using virtualization

Now we need to configure netboot.xyz
